# Discord Server Tracker

Discord Server Tracker — это расширение для браузера, предназначенное для отслеживания взаимодействий пользователей с кнопками присоединения к серверам Discord. Оно предоставляет удобный способ логировать клики, вести историю посещений серверов и отображать информацию в удобном интерфейсе всплывающего окна.

---

## Возможности

- **Отслеживание серверов**: Считает количество кликов на кнопках присоединения к серверу.
- **История серверов**: Сохраняет записи о посещённых серверах, включая их названия, ссылки для присоединения и время последнего посещения.
- **Интерфейс всплывающего окна**: Предоставляет чистый, постраничный интерфейс для просмотра отслеживаемых серверов с возможностью поиска, навигации и сброса данных.
- **Периодическое обновление**: Автоматически обновляет данные при добавлении новых серверов или взаимодействиях с ними.
- **Постоянное хранилище**: Сохраняет все данные локально с использованием API хранилища браузера.
- **Мультисайтовая поддержка**: Помимо [server-discord.com](https://server-discord.com), расширение теперь поддерживает также [myserver.gg](https://myserver.gg).
- **Экспорт и импорт данных**: Возможность экспорта данных о серверах в JSON-файл и их импорта для резервного копирования или переноса.
- **Многоязычный интерфейс**: Поддержка выбора языка интерфейса через настройки (на данный момент поддерживаются английский, русский и другие языки через файлы локализации).
- **Интерактивная кнопка сброса**: Кнопка для сброса счетчиков с необычным поведением — меняет форму, позицию и реагирует на движение курсора, добавляя элемент игры при использовании.
- **Динамическое обновление интерфейса**: При изменении языка или обновлении данных интерфейс моментально подстраивается под актуальные настройки и данные.

> **Примечание**: Это расширение работает исключительно на сайтах [https://server-discord.com](https://server-discord.com) и [https://myserver.gg](https://myserver.gg).

## Установка

### Установка расширения локально

1. **Клонируйте или скачайте репозиторий**  
   Склонируйте данный репозиторий на ваш локальный компьютер или скачайте его в виде ZIP-архива и извлеките файлы.

2. **Перейдите на страницу управления расширениями**  
   Откройте Chrome или любой браузер на основе Chromium и перейдите на страницу управления расширениями:

   ```
   chrome://extensions/
   ```

3. **Включите режим разработчика**  
   Включите переключатель "Режим разработчика" в правом верхнем углу страницы.

4. **Загрузите расширение**  
   Нажмите кнопку "Загрузить распакованное расширение" и выберите папку с файлами расширения.

5. **Проверьте установку**  
   Расширение должно появиться на панели расширений вашего браузера. Кликните по нему, чтобы открыть интерфейс всплывающего окна.

### Установка через Chrome Web Store

Вы также можете установить расширение напрямую из [Chrome Web Store](https://chromewebstore.google.com/detail/discord-server-tracker/ncjajonogknecpjaknfimackajgfbjhb?hl=en&authuser=0). Просто перейдите по ссылке и нажмите "Установить".

## Использование

1. Перейдите на сайт [https://server-discord.com](https://server-discord.com) или [https://myserver.gg](https://myserver.gg).
2. Взаимодействуйте с кнопками присоединения к серверам Discord на сайте.
3. Откройте всплывающее окно расширения, чтобы просмотреть данные о серверах, включая количество кликов, историю посещений и дополнительные возможности:

- Используйте поиск и навигацию для быстрого доступа к нужным серверам.
- Экспортируйте текущие данные в JSON-файл для создания резервной копии.
- Импортируйте ранее сохранённые данные, чтобы восстановить историю.
- Меняйте язык интерфейса в настройках для более комфортного использования.
- Сбросьте все счетчики, используя интерактивную кнопку сброса (удерживайте Ctrl при клике для подтверждения сброса).
