Privacy Policy for Discord Server Tracker

Last updated: July 18, 2026

Your tracked server data stays on your device. Discord Server Tracker does not operate a data-collection server, use analytics, or sell or share user data.

Overview

Discord Server Tracker is a browser extension that helps users track and organize their interactions with Discord server listings on supported websites.

This Privacy Policy explains what information the extension processes, why it is processed, where it is stored, and how users can delete it.

Supported Websites

Discord Server Tracker operates only on the following websites:

  • server-discord.com
  • myserver.gg
  • discordserver.info
  • disboard.org

Information Processed

When a user clicks a supported Discord server's Join button, the extension may process and store:

  • The Discord server name and identifier.
  • The server listing page URL.
  • The server Join URL.
  • The number of Join-button clicks.
  • The dates and times of Join-button clicks.
  • The date and time of the first and most recent interaction.
  • Favorite and archived status.
  • Notes and tags entered by the user.
  • Extension preferences, including language, theme, and time format.

The extension processes only the information necessary to provide its server tracking and organization features.

How Information Is Used

The processed information is used exclusively to:

  • Display the user's tracked Discord servers.
  • Show Join-button click counts and interaction history.
  • Search, sort, filter, favorite, archive, and organize tracked servers.
  • Store personal notes and tags.
  • Export and import tracked server records.
  • Remember extension preferences.

The information is not used for advertising, profiling, analytics, creditworthiness assessment, or any unrelated purpose.

Local Storage

All extension data is stored locally on the user's device using the Chrome Storage API.

Discord Server Tracker does not transmit tracked server records, browsing activity, notes, settings, or other user data to the developer's servers or any external service.

The extension does not use Chrome Storage Sync.

Data Sharing

Discord Server Tracker does not sell, rent, transfer, or share user data with third parties.

The developer does not receive or have access to the information stored by the extension.

Analytics and Advertising

Discord Server Tracker does not use:

  • Analytics services.
  • Advertising services.
  • Tracking pixels.
  • External telemetry.
  • Behavioral profiling.
  • Personalized advertising.

Remote Code

Discord Server Tracker does not execute remotely hosted JavaScript or WebAssembly.

All executable code is included in the extension package distributed through the Chrome Web Store.

Data Export and Import

Users may export their tracked server records as a JSON file.

Exported files are created locally on the user's device. The extension does not upload exported or imported files to any server.

Users are responsible for securely storing any exported files.

Data Retention and Deletion

Information remains stored locally until the user:

  • Deletes individual tracked server records.
  • Uses the extension's reset functionality.
  • Clears the extension's browser storage.
  • Uninstalls the extension.

Because the developer does not receive or store this information remotely, the developer cannot access, recover, or delete it on the user's behalf.

Security

User data remains within the browser's local extension storage and is not transmitted over the Internet by Discord Server Tracker.

Users should protect exported JSON files as they would any other locally stored personal file.

Chrome Web Store Limited Use

The use of information received from Chrome APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Information is processed only to provide or improve the user-facing functionality of Discord Server Tracker.

Changes to This Privacy Policy

This Privacy Policy may be updated when the extension's functionality or data-handling practices change.

Updates will be published on this page, and the “Last updated” date will be revised.
